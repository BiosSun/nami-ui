ListBox

用于从一组数据中选择一条或多条数据。

多选#

通过设置 multiple 属性，可以开启多选，此时 value 需要改为数组类型。

显示单/复选框#

通过设置 check 属性，可以在每个选项前显示一个单/复选框（根据是否多选自动切换）。
